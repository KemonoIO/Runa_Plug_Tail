Runa Plug Tail

Summary

This package is a XLR-esque plug tail addon for the Runa model by Torinyan.

Dependencies

Required

  • VRCSDK3-Avatar
  • Runa Base Model

Optional

  • Poiyomi Toon 8.1.160

Installation

  1. Unpack the dependencies, in order.
    1. Only unpack Poiyomi Toon if you intend to alter the tail material.
  2. Drag Assets/Kemono_Io/Runa_Plug_Tail.prefab into the Runa_Source object.
  3. Constrain the Runa_Plug_Tail object to the hips.
    1. Select the Runa_Plug_Tail object in the Hierarchy panel.
    2. Drag the hips bone into the Parent Constraint's Source in the Inspector panel.
    3. Click the "Activate" button.
  4. Set the Runa_Plug_Tail's anchor override.
    1. Select the Runa_Plug_Tail mesh in the Hierarchy panel.
    2. Drag the hips bone into the Anchor Override in the Inspector panel.
