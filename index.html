Installation

  1. Unpack the dependencies, in order.
  2. Drag Assets/Kemono_Io/Runa_Plug_Tail.prefab into the Runa_Source object.
  3. Constrain the Runa_Plug_Tail object to the hips.
    1. Select the Runa_Plug_Tail object in the Hierarchy panel.
    2. Drag the hips bone into the Parent Constraint's Source in the Inspector panel.
    3. Click the Activate button.
  4. Set the Runa_Plug_Tail's anchor override.
    1. Select the Runa_Plug_Tail mesh in the Hierarchy panel.
    2. Drag the hips bone into the Anchor Override in the Inspector panel.
  5. Add the tail to the emission and hue shift animations.
    1. Select the Runa_Source object in the Hierarchy panel.
    2. Set the Animator's Controller to FX_Runa in the Inspector panel.
    3. Select the Animation tab and click the lock icon if it is not already locked.
    4. Select the color_0 animation and click the record button.
    5. Set the Hue Shift property to 0.
    6. Select the color_1 animation and click the record button.
    7. Set the Hue Shift property to 1.
    8. Select the body_lightning_0 animation and click the record button.
    9. Set the Emission Strength property to 0.
    10. Select the body_lightning_1 animation and click the record button.
    11. Set the Emission Strength property to 0.6.
    12. Click the Preview button to exit record mode.
    13. Click the lock icon.
    14. Right click the Animator's Controller and select Revert.
